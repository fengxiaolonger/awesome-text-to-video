# Contributing to Awesome Text-to-Video

Thank you for your interest in contributing! Here's how you can help.

## Adding a Tool

1. Fork this repository
2. Add the tool in the appropriate category
3. Follow the existing table format
4. Submit a pull request

### Requirements

- The tool must be publicly available (not in private beta)
- Include accurate pricing and feature information
- Tools should be specifically related to AI video generation
- No duplicate entries

### Format

For table entries, use:

```markdown
| [Tool Name](https://url.com) | Description | Feature | Rating |
```

## Updating Information

If you notice outdated information, please submit a PR with the correction. Include a source or reference when possible.

## Reporting Issues

- Broken links
- Inaccurate information
- Missing important tools
- Spam or inappropriate content

Please [open an issue](https://github.com/fengxiaolonger/awesome-text-to-video/issues/new) with details.

## Code of Conduct

Be respectful and constructive. This is a community resource for everyone.
